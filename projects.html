Ecommerce APP Admin

Description:

This is a Spring Boot Ecommerce Admin APP
This app consists of multiple modules like Users, Categories, Brands, Products, Reviews, Questions, Customers, Shipping, Orders, Settings and more. Each module makes changes to the main e-commerce application. Each module performs CRUD operations, for example Users Module as admin I can see all users, add a new user, update a user and delete a user, as well as search a user by first and last name or sort the list of users. This application has been secured with Spring Securiy, so you need an account to access it.
This application was developed using Spring Boot, Spring Data JPA, Spring MVC, Spring Security, Thymeleaf, CSS MySQL DB and Jquery and deployed on Heroku Cloud Platform. Please wait a few seconds for the app to load.
